Chattr Demo

This page demonstrates the Chattr embeddable chatbot widget. Click the chat bubble in the bottom-right corner to start a conversation.

To embed Chattr on your own site, add this script tag: 

<script
  src="https://your-server.com/widget.js"
  data-server="https://your-server.com"
  data-theme-primary="#e27d60"
  data-theme-title="Ask me anything"
  defer
></script>
